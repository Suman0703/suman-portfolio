## ✨ Key Features

- **Custom Boot Sequence:** A personalized, animated SVG loading screen that simulates a terminal booting up before revealing the portfolio.
- **Terminal UI Aesthetic:** Features a strict dark mode (`#0a0a0f`) with high-contrast neon green (`#00FF99`) highlights and monospace typography.
- **Interactive Accordions:** Custom-built expandable achievement cards using standard state management for a clean, zero-clutter UI.
- **Live Data Integrations:** Features a real-time ticking clock component and direct web-mail integration to bypass default desktop mail clients.
- **Fully Responsive:** Fluid layouts that perfectly adapt from ultra-wide desktop monitors down to narrow mobile screens using optimized Flexbox and Grid structures.

# Installation
### 1. Clone the repo
```sh
git clone [https://github.com/Suman0703/portfolio.git](https://github.com/Suman0703/portfolio.git)
```

### 2. Navigate into the directory
```sh
cd portfolio
```
### 3. Install NPM packages

```sh
npm install
```
### 4. Start the development server
```sh
npm run dev
```
